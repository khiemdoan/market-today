# Market Today

Automatically send crypto notifications to Telegram using GitHub Actions.

Join Telegram group: [Kênh lùa gà](https://t.me/kenh_lua_ga)

## Setup environment

Create virtual environment

```bash
uv venv
```

Check dependencies

```bash
uv lock -v -U
```

Install dependencies

```bash
uv sync -v --group tools --group testing --group linting
```
